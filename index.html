Как это работает

  1. Создаете офферы в админ-панели
  2. Для каждого оффера указываете на каких лендингах он будет отображаться
  3. Данные сохраняются в localStorage браузера
  4. Лендинги загружают офферы из localStorage с помощью API ключа
  5. Каждый лендинг видит только свои офферы (по полям landing1, landing2)
